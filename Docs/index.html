BtbTweet - Free Twitter Module

This DNN (DotNetNuke) module can be used to display tweets from Twitter on your site. This simple and lightweight module can be customise using one of the three the built in templates or you can create your own.

Simple Setup

Two modes can be used to select the tweets that are displayed in the module.

  • Display tweets from a specific user's timeline
  • Display tweets based on a search

Results are cached for 15 minutes to improve performance.

Template Based

The results are rendered using a template, the modules has three templates out of the box, or you can get creative and design your own.

  • Angular Template, AJAX template
  • React Template, AJAX template
  • Razor Template

The templates are located in the folder `DesktopModules/BtbTweet/Templates`.

To create a new template create a razor file in the folder then in the module settings pick the new file to use to output the data.
